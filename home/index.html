This class is primarily to learn, design, and create websites.

Assignments

  1. Assignment 1: Basic HTML

    This assignment was to create a website on a topic of choice. I chose Ferrari F1.

  2. Assignment 2: Basic CSS

    This assignment was to create a webpage using rows and columns as well as incorporating CSS into the HTML. I used McLaren F1 to incorporate images and a color theme.

  3. Assignment 3: Page Layout

    This assignment is to create a flex layout with links and different colors as well as a theme and make it matching. It also needs placeholder images and Lorem Ipsum. I used UofSC colors to start the creation of my webpage.

  4. Assignment 4: Updated Home Page

    This assignment is to add links and screenshots and update my webpage.

  5. Assignment 5: CSS Page Recreate

    This assignment is to recreate a CSS Page using CSS and HTML.

  6. Assignment 6: Main Page Projects Section

    This assignment is to add a separate projects section to my webpage.

  7. Assignment 7: JavaScript, Buttons, Functions, and More

    This assignment is to use HTML and CSS to create a page.

  8. Assignment 8: Conditionals

    This assignment is creating 2 pages using one html file. Two exercises on one page.

  9. Assignment 9: Loops

    This assignment is creating 2 pages using one html file. Two exercises on one page. A loops assignment.

  10. Assignment 10: Arrays

    This assignment is using javascript to create a banner which switches ads every 2 seconds with an array of images.
